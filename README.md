# 🚗 Parking Lot Management System


## Project Overview

The **Parking Lot Management System** is built to simulate the core operations of a parking facility. It allows the database to:

- store customer information
- associate customers with vehicles
- manage parking spots
- track vehicle entry and exit through tickets
- record payments for parking sessions

This project emphasizes **data integrity**, **clear relationships**, and **normalized design**.
